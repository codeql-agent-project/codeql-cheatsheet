Jump-to-definition links

java

Author: Semmle LGTM

import definitions
external string selectedSourceFile();
from Element e, Element def, string kind
where def = definitionOf(e, kind) and e.getFile() = getFileBySourceArchiveName(selectedSourceFile())
select e, def, kind

Description

Generates use-definition pairs that provide the data for jump-to-definition in the code viewer.
